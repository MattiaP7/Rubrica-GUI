Detailed Description

The main application window class.

La classe della finestra principale dell'applicazione

This class represents the main window of the application and handles all user interface interactions and contact management operations.

Questa classe rappresenta la finestra principale dell'applicazione e gestisce tutte le interazioni con l'interfaccia utente e le operazioni di gestione dei contatti.
